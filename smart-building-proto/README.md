# 智慧楼宇小程序高保真原型 (Smart Building Prototype)

这是一个基于 React + Tailwind CSS 构建的智慧楼宇小程序高保真原型。

## 核心功能

- **首页 (Home)**: 3D 风格轮播图、实时能源监控、天气预报。
- **招商 (Leasing)**: 物业列表展示，内置 **租金回报率计算器**。
- **名企 (Enterprise)**: 楼宇入驻企业品牌墙。
- **物业 (Property)**: 报事报修、停车缴费、入驻申请等服务入口。
- **钱包 (Wallet)**: 虚拟一卡通，支持 **模拟支付场景动画**。
- **角色切换**: 实时切换 “业主” 与 “租客” 视角，展示不同数据。

## 技术栈

- **框架**: React 19 (Vite)
- **样式**: Tailwind CSS v4 (玻璃拟态、渐变色设计)
- **动画**: Framer Motion (平滑页面过渡与交互动画)
- **图标**: Lucide React

## 快速开始

1. 安装依赖:
   ```bash
   npm install
   ```

2. 启动开发服务器:
   ```bash
   npm run dev
   ```

3. 构建生产版本:
   ```bash
   npm run build
   ```
